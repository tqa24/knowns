# Web UI

Knowns có giao diện web để làm việc trực quan với task, doc, graph, config, và chat.

## Mở

```bash
knowns browser
knowns browser --open
```

## Các khu vực chính

- Board và task views
- Doc browser
- Graph / knowledge views
- Config pages
- Chat

## Khi nào nên dùng

- Xem task theo kiểu board
- Duyệt doc tiện hơn CLI
- Explore graph hoặc dùng chat-driven workflows
