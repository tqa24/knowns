# Guidance files

Knowns dùng một file canonical và nhiều compatibility files cho các AI runtimes.

## Canonical

- `KNOWNS.md`

## Compatibility files

- `CLAUDE.md`
- `OPENCODE.md`
- `GEMINI.md`
- `AGENTS.md`
- `.github/copilot-instructions.md`

## Refresh

```bash
knowns sync
knowns sync --instructions
knowns agents --sync
```

Không còn command riêng `knowns guidelines` nữa.
