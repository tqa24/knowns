# Guidance files

Knowns dùng một file canonical và nhiều compatibility files cho các AI runtimes.

## Canonical

- `KNOWNS.md`

## Compatibility files

- `CLAUDE.md`
- `OPENCODE.md`
- `GEMINI.md`
- `AGENTS.md`
- `.github/copilot-instructions.md`

## Refresh

```bash
knowns setup
knowns sync
knowns sync --instructions
```

Dùng `knowns setup` để tạo platform files lần đầu, hoặc `knowns sync` để refresh.
