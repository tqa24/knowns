# Web UI

Knowns includes a browser UI for visual interaction with tasks, docs, graph views, config, and chat.

## Open it

```bash
knowns browser
knowns browser --open
```

## Main areas

- board and task views
- docs browser
- graph / knowledge views
- configuration pages
- chat page

## When to use it

- when you want a board-oriented task view
- when browsing docs is easier in a UI than in CLI output
- when you want graph exploration or chat-driven workflows
