# Knowns Documentation

Knowns is a project context layer for software teams and AI agents. It keeps project tasks, docs, memory, templates, search, and agent integrations in one repo-local system so people and AI can work from the same source of truth.

If you are new to Knowns, start with installation, then run the quick start, then create your first project context. The guides explain the daily workflow after that.

## Notes

- `docs/en/` is the primary maintained public documentation set.
- `docs/vi/` mirrors the same structure as a Vietnamese translation layer.
- Runtime-critical agent guidance lives in Knowns MCP `initial` and on-demand `help`; repo files such as `AGENTS.md` are lightweight compatibility shims for runtimes that auto-detect instruction files.
